# VuePress+GitHub+TravisCI 持续部署个人网站

VuePress 1.x 静态网站生成器

GitHub 开源及私有软件项目的托管平台

TravisCI 开源持续集成构建项目

# GitHub 准备

如果不想缴费那就GitHub上需要需要创建两个repo，一个存放文档工程，作为GitHub Page源。

# 注册GitHub账号

https://github.com/join

创建完成之后登陆GitHub

# 创建文档工程仓库

创建文档工程仓库，在此工程里进行文档编写工作

# 点击右上角+--new repository

创建名为仓库，比如叫做doc-test

点击按钮create repository

创建完成之后先不用初始化，等一下通过我们创建的本地工程进行初始化。

# 创建<username>.github.io仓库

以我个人的账号youlingred为例

同上，创建仓库，仓库名为 youlingred.github.io

勾选Initialize this repository with a README

点击按钮create repository创建完成

# 添加 SSH key

点击左上角用户头像，点击setting

点击SSH and GPG keys--> New SSH key

注意

如何生成SSH key，去这里查看

# 生成GITHUB_TOKEN

我们这里先生成token，下面会用到

点击左上角用户头像，点击setting，选择左侧菜单

选择Personal access tokens

点击Generate new tocken

生成的token，一定要拷贝，保存起来

# VuePress 搭建文档工程

# 环境配置

安装NodeJS,Git

安装cnpm淘宝镜像

$ npm i -g cnpm --registry=https://registry.npm.taobao.org

# 创建工程文件夹

# 新建文件夹doc-test
$ md doc-test

#进入doc-test目录
$ cd doc-test

# 工程初始化

# npm初始化工程，生成package.json文件
$ npm init

# 安装Vuepress 1.x

# 安装Vuepress
$ cnpm i vuepress@next -D

# 修改package.json脚本scripts

{
  "scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}

# 添加基础文件

# 新建文件夹docs
$ md docs

# 新建一个 markdown 文件
$ echo '# Hello VuePress!' > docs/README.md

# 本地运行

$ npm run dev

可以看到网站运行了

# 初始化文档工程仓库

工程创建好了，那么我们现在就用工程初始化git仓库

现添加.gitignore忽略文件，文件内容如下

**/.idea
**/.DS_Store
**/.vscode
**/dist
**/node_modules
npm-debug.log*
yarn-debug.log*
yarn-error.log*

接着继续初始化工程

# git初始化
$ git init

# 添加文件版本控制
$ git add --all

# 提交到本地仓库
$ git commit -m 'init'

# 添加remote地址 git@github.com:youlingred/doc-test.git 替换成你自己的地址
$ git remote add origin git@github.com:youlingred/doc-test.git

# 推送到远程仓库
$ git push --set-upstream origin master

远程仓库初始化

# 持续部署配置

# 添加travis.yml 部署文件

进入本地工程目录，新建travis.yml，写入下面文件内容

这里我们将用到之前保存的GITHUB_TOKEN

language: node_js
node_js:
  - lts/*
branches:
  only:
    - master
script:
  - npm run build
deploy:
  provider: pages
  skip-cleanup: true
  local_dir: docs/.vuepress/dist
  # 要推送到的git仓库地址，替换成你自己的xxx/xxx.github.io
  repo: youlingred/youlingred.github.io
  target_branch: master
  # $GITHUB_TOCKEN 替换成我们上面保存的GITHUB_TOKEN
  github-token: $GITHUB_TOCKEN
  keep-history: true
  on:
    branch: master

# 通过GitHub账号登录TravisCI

# 进入setting页面

# 选择并添加需要构建的仓库

点击Manage repositories on GitHub配置需要部署的工程

就可以看到我们的仓库已经添加到travis工程部署列表了
